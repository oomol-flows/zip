# ZIP

[English](./README.md) | Chinese

## 简介

ZIP 是一个基于 `adm-zip` 封装的 OOMOL Block，用于简化文件和文件夹的压缩与解压操作。通过该模块，用户可以轻松实现将文件或文件夹压缩为 ZIP 格式，或者将 ZIP 文件解压至指定位置。

## 功能

- **压缩文件或文件夹**：支持将单个文件或整个文件夹打包为 ZIP 格式的压缩文件。
- **解压 ZIP 文件**：支持从 ZIP 压缩包中提取文件至指定目录。
- **灵活的操作选项**：提供多种配置选项，例如指定输出路径、包含或排除特定文件等。
- **易于集成**：作为 OOMOL Block 的一部分，可以无缝集成到更大的工作流或项目中。

## 注意事项

- 在压缩大文件或文件夹时，请确保系统有足够的磁盘空间。
- 解压前请确认目标目录是否为空，避免文件覆盖导致的数据丢失。
- 支持自定义配置项以满足不同场景需求，具体可参考 `adm-zip` 官方文档。
